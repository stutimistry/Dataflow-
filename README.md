Next.js + TanStack Query — Advanced Data Management 🚀

A modern data management application built with Next.js App Router and TanStack Query featuring SSR-ready architecture, pagination, infinite scrolling, caching, CRUD operations, optimistic updates, and debounced search.

✨ Features
✅ Server-Side Rendering (SSR) Ready
✅ Client-Side Caching with TanStack Query
✅ Pagination
✅ Infinite Scrolling
✅ CRUD Operations (Create/Delete)
✅ Optimistic Updates
✅ Query Invalidation
✅ Debounced Search
✅ Loading & Error Handling
✅ Background Refetching
✅ TanStack Query Devtools
✅ Clean & Modular Folder Structure
✅ Responsive UI with Tailwind CSS
